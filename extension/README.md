# Monolithic Curator - Browser Capture Utility

## Installation Guide

1. **Extract this Zip**: Unzip this folder to a permanent location on your computer.
2. **Open Extensions**: Open Google Chrome and navigate to `chrome://extensions`.
3. **Enable Developer Mode**: Toggle the "Developer Mode" switch in the top-right corner.
4. **Load Unpacked**: 
   - Click the **"Load unpacked"** button.
   - Select this folder (the one containing `manifest.json`).
5. **Pin to Toolbar**: Click the puzzle icon in your Chrome toolbar and pin **Brain Cache** for easy access.

## How to Use

- **Capture**: Click the Brain Cache icon on any webpage to save it to your neural library.
- **Auto-Sync**: If you are logged into the web app, captures are automatically associated with your account.
- **Metadata**: The AI will automatically tag and categorize your captures in the background.
